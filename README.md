#  EasyInvest – Startup Investment Platform
EasyInvest is a full-stack investment platform that connects startup founders with investors. The system allows founders to showcase their startups and investors to discover, filter, and invest based on their preferences.
##  Overview
This project is designed to simulate a real-world investment ecosystem with role-based access, secure transactions, and intelligent matching between investors and startups.
##  User Roles
###  Founder
- Create and manage startup profiles  
- Add funding requirements and business details  
- Track investor interest  
###  Investor
- Browse and filter startups  
- Invest in startups  
- Track investment history  
###  Admin
- Manage users and platform data  
- Monitor platform activity  
##  Features
- Role-based authentication (Admin / Founder / Investor)  
- JWT-based secure login system  
- Startup listing and filtering  
- Investment system (one investment per startup per investor)  
- Investor interest system  
- Funding progress tracking  
- Dynamic industry/category support  
## Architecture
- Backend:
  - Controller → Service → Repository  
- Frontend:
  - Angular-based UI (in progress)
